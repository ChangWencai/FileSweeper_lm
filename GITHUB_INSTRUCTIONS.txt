# FileSweeper - 重复文件管理工具

## 推送代码到GitHub的步骤

1. 在GitHub网站上创建一个新的仓库（例如命名为FileSweeper）

2. 添加远程仓库地址：
   ```bash
   git remote add origin https://github.com/你的用户名/FileSweeper.git
   ```

3. 推送代码到GitHub：
   ```bash
   git push -u origin master
   ```

## 注意事项

- 如果你启用了两步验证，需要生成个人访问令牌(Personal Access Token)代替密码
- 确保你的GitHub账户已经配置好SSH密钥或者使用HTTPS方式连接
- 如果遇到网络问题，可以尝试使用代理或者修改DNS设置

## 项目结构说明

- `main.py` - 程序入口文件
- `core/` - 核心功能模块（文件扫描、重复文件检测）
- `ui/` - 用户界面相关代码
- `requirements.txt` - 项目依赖列表
- `README.md` - 项目说明文档
- `.gitignore` - Git忽略文件配置

## 依赖安装

在部署到新环境时，使用以下命令安装依赖：
```bash
pip install -r requirements.txt
```

## 运行程序

使用以下命令运行程序：
```bash
python main.py
```
